Simon Says Game

Simon Says Memory Game - Challenge Your Brain!

How to Play

  • Press any key to start the game.
  • Watch the color that flashes; remember the sequence.
  • Click the buttons in the same order as they flashed.
  • If you click the correct sequence, you move to the next level.
  • If you make a mistake, the game ends, and you can restart.
